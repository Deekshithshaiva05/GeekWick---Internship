Basic Tasks 
1. Create and switch to a database called internhub. 

> use internhub


2. Create a collection called students. 

> db.createCollection("students")


3. Insert 1 document into students with fields: name, age, course. 

db.students.insertOne({
  name: "Deekshith",
  age: 21,
  course: "AI & ML"
})


4. Insert 1 more document with different values. 

db.students.insertOne({
  name: "Vinay",
  age: 22,
  course: "Computer Science"
})


5. Show all documents from students.

> db.students.find()

> db.students.find().pretty()

4. Compare the students collection structure with how it would look in MySQL.

> MongoDB (Document-Based)

  "name": "Deekshith",

}

> MySQL (Table-Based)
CREATE TABLE students (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(50),
  age INT,
  course VARCHAR(50)
);
