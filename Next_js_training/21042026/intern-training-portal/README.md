# 🚀 Intern Training Portal (Next.js)

This project is a **Next.js-based web application** developed as part of my internship training at **Geekwick TechMedia Services Pvt Ltd**.  
It demonstrates routing, reusable components, nested layouts, and UI enhancements using modern React and Next.js concepts.

---

## 📌 Features

- ✅ Next.js App Router (File-based routing)
- ✅ Multiple Pages:
  - Home
  - About
  - Contact
  - Services
  - Courses
  - Trainers
- ✅ Dashboard with Nested Routing:
  - Profile Page
  - Settings Page
- ✅ Reusable Components:
  - Header
  - Footer
  - Card
  - CourseCard
- ✅ Course Listing with Search Functionality 🔍
- ✅ Conditional Rendering (No Courses Found)
- ✅ Sticky Footer Layout
- ✅ Active Navigation Highlighting
- ✅ Dark/Light Theme Toggle 🌙☀️
- ✅ Global CSS Styling

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```
git clone https://github.com/beingkaushik/geekwick_internship.git
```
### 2️⃣ Navigate to project folder
```
cd training/Nextjs/intern-training-portal
```
### 3️⃣ Install dependencies
```
npm install
```
### 4️⃣ Run development server
```
npm run dev
```
### 5️⃣ Open in browser
```
http://localhost:3000
```
